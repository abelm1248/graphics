Load the am73272_assignment2.pde file and click "run" to start the program. A sample image depicting a mountain will be displayed to demonstrate the following filters.

Pressing 1 will apply the grayscale filter to the image.

pressing 2 will increase the contrast of the image.

pressing 3 will blur the image.

pressing 4 will run edge detection on the image.

pressing 0 will reset the image to its original state.
